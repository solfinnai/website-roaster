# Website Roaster 🔥

A viral lead-gen tool that gives users brutally honest (but constructive) reviews of their websites.

## What It Does

1. User pastes any URL
2. Shows an "analyzing" animation
3. Generates a randomized roast with scores for:
   - Design 🎨
   - Speed ⚡  
   - Mobile 📱
   - SEO 🔍
4. Delivers brutal but funny feedback
5. CTAs for booking consultation or redesign services

## Features

- **Single HTML file** - No build process needed
- **Dark theme** with neon red/orange accents
- **Randomized roasts** - Feels dynamic even without real API calls
- **Mobile responsive** design
- **Social sharing** functionality
- **Lead generation** CTAs
- **Fun, viral tone** designed for social media sharing

## Design

- **Fonts**: Inter + Space Grotesk
- **Colors**: Dark background with red/orange/green neon accents
- **Tone**: Irreverent, roast comedy show vibe
- **Emojis**: Flames everywhere 🔥

## Deployment

### Vercel (Recommended)
1. Upload `index.html` to a new Vercel project
2. Deploy (auto-deploys from git or direct upload)
3. Update social meta tags with your actual domain

### Any Static Host
- GitHub Pages
- Netlify  
- AWS S3 + CloudFront
- Even basic web hosting

## Customization

To customize for your business:

1. **CTAs**: Update the cal.com link and redesign email
2. **Branding**: Change "Website Factory" to your company name
3. **Social Tags**: Update og:image and og:url meta tags
4. **Domain**: Update all references from website-roaster.vercel.app

## How The "AI" Works

Since we can't call real AI APIs from frontend without exposing keys:

- Uses pre-written comment templates
- Randomizes scores within realistic ranges  
- Picks random comments for each category
- Creates the illusion of personalized analysis
- Still provides genuine value through structured feedback

## Viral Elements

- **Shareable results** that work on social media
- **Fun, memorable experience** people want to try on friends' sites
- **Clear value proposition** for the target audience
- **Low friction** - just paste URL and go

1. **Free tool** drives traffic and engagement

3. **CTAs** convert to consultations and redesign services
